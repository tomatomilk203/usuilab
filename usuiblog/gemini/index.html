Geminiとは？

GeminiはGoogleが作ったAIです。2023年に登場し、2024年に「Gemini（ジェミニ）」という名前になりました。

「Google検索」「Gmail」「Googleマップ」を作った会社が、同じGoogleのサービスと繋がるAIを作った——それがGeminiの最大のポイントです。

Gmailも、Googleドキュメントも、Googleマップも、全部AIに手伝ってもらえる時代。

Androidスマホを使っている人は、すでにGeminiが入っているかもしれません。Googleアシスタントの後継として、多くのスマホに標準搭載されています。

ChatGPTと何が違うの？

よく「ChatGPTとどっちがいいの？」と聞かれます。どちらも「会話できるAI」ですが、得意なことが少し違います。

ChatGPT
→ OpenAI製
→ 文章・アイデア出しが得意
→ プラグインで機能拡張
→ Googleサービスとは別
Gemini
→ Google製
→ 検索・最新情報が得意
→ Gmail・Docsと連携
→ Androidに標準搭載
どっちを使えばいい？

正直、どちらか一方だけ使えば十分です。Googleのサービスをよく使う人はGemini、そうでない人はChatGPTが馴染みやすいかもしれません。両方無料で試せるので、使い比べてみるのがいちばんです。

何ができるの？

基本的な会話・質問・文章作成はChatGPTと同じくできます。Geminiならではの強みはこちら。

  • 🌐
    最新情報を調べられる Google検索とつながっているので、今日のニュースや最近の出来事も答えてくれます。ChatGPTの無料版は知識に時間的な制限がありますが、Geminiはリアルタイムで検索できます。
  • 📧
    Gmailのメールを読んで要約・返信 「今週届いたメールで大事なものを教えて」「このメールに返信を書いて」が、Gmailと連携してできます（有料版）。
  • 📄
    Googleドキュメントで文章を書く 「提案書の下書きを書いて」「この文章を丁寧な言葉に直して」など、ドキュメントを開きながらそのまま使えます（有料版）。
  • 🖼️
    写真を見て答えてくれる スマホで写真を撮って「これ何？」「このレシピを教えて」と聞くと、画像を見ながら答えてくれます。
  • 🗺️
    Googleマップと連携（一部対応） 「渋谷でランチのおすすめ教えて」と聞くと、マップと連携して候補を出してくれる機能が追加されています。

実際の使いどころ

「機能は分かった、でも実際どう使うの？」という人のために、日常のシーンを紹介します。

📬
メールの返信が面倒なとき 「このメールに、丁寧にお断りする返信を書いて」とGeminiに頼むと、文章を作ってくれます。あとは自分で確認して送るだけ。
📋
会議の議事録をまとめたいとき メモをGeminiに貼り付けて「要点をまとめて」と頼むだけ。そのままGoogleドキュメントに書き出すこともできます。
📸
スーパーで食材の写真を撮ったとき 「この食材でできる夕食レシピを3つ教えて」と写真を送るだけ。冷蔵庫の中を撮っても同じことができます。
🔍
最新ニュースを調べたいとき 「今週のAIに関するニュースを教えて」のように、今日の情報を含めた質問ができます。Google検索とつながっているので情報が新しい。

どうやって使うの？

パソコンでもスマホでも、どちらからでも始められます。

01
Googleアカウントを用意する GmailやYouTubeに使っているGoogleアカウントがあればOK。持っていなければ無料で作れます。
02
パソコンなら gemini.google.com にアクセス ブラウザを開いてアドレスを入力するだけ。Googleアカウントでログインして、すぐ使えます。
03
スマホなら「Gemini」アプリをインストール App Store / Google Playで「Gemini」で検索。AndroidはGoogleアシスタントの代わりにそのまま使えるケースも。
04
話しかけてみる 「こんにちは」から始めてOK。文章でも音声でも対応しています。
まずこれを試してみて

「今日の東京の天気を教えて、それと週末の予定を立てるのを手伝って」と話しかけてみてください。Geminiはリアルタイム検索を使いながら答えてくれます。

無料で使える？有料版との違いは？

Geminiも無料で使えます。有料版は「Gemini Advanced」といい、Googleの最上位モデルが使えます。

無料版 Gemini
✓ 基本的な会話・質問
✓ 最新情報の検索
✓ 画像の読み取り
✓ 文章作成・翻訳
— Gmail・Docs連携は限定的
有料版 Gemini Advanced
✓ 最上位AIモデル
✓ Gmail完全連携
✓ Googleドキュメント連携
✓ より長い会話ができる
✓ Google Oneの2TBストレージ付き

有料版はGoogle Oneのプランに含まれていて、月2,900円程度です。Gmail連携などを仕事で使いたい人には価値があります。まずは無料版で試してみましょう。

注意点

Geminiも万能ではありません。使う前に知っておいてほしいことを伝えます。

⚠️
間違いをもっともらしく答えることがある

Google検索と連携しているとはいえ、Geminiも誤った情報を自信満々に話すことがあります。医療・法律・お金に関する大事な判断は、必ず専門家や公式サイトで確認してください。

入力した内容はGoogleに送られる

Geminiに入力した内容は、Googleのサーバーに送られます。個人情報・会社の機密情報・パスワードなどは絶対に入力しないようにしましょう。設定でデータの保存をオフにすることもできます。

Googleアカウントと連携するからこそ注意

GmailやGoogleドライブと連携できるのはGeminiの強みですが、それはGeminiがあなたのデータにアクセスできるということでもあります。どの機能を許可するかは、設定画面で確認・変更できます。

この記事のまとめ
  • GeminiはGoogleが作ったAI。Androidスマホには標準搭載されていることも
  • 最大の強みは「Google検索・Gmail・Docsとの連携」
  • 最新情報を検索しながら答えられるのはChatGPT無料版との大きな違い
  • gemini.google.com またはアプリで、Googleアカウントがあれば今すぐ無料で使える
  • Gmail・Docsの本格連携は有料版（Gemini Advanced）が必要
  • 間違いを自信満々に言うことがある。大事なことは必ず確認する
  • 個人情報や機密情報は入力しない
